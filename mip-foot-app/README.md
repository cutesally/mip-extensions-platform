# mip-foot-app

mip-foot-app 组件说明
打开app和下载app
标题|内容
----|----
类型|通用
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|https://c.mipcdn.com/static/v1/mip-foot-app/mip-foot-app.js

## 示例

### 基本用法
```html
<mip-foot-app>
<div class="Mfoot_app"  id="downloadApp">
			下载APP</div>
<div id="openApp" >打开APP</div>
</mip-foot-app>
```

## 属性

### topicID

说明：主贴id
必选项：否
类型：整数
取值范围：无
单位：无
默认值：0

### replyID
说明：主贴id
必选项：否
类型：整数
取值范围：无
单位：无
默认值：0

## 注意事项
下载app的标签id为downloadApp
打开app的标签为openApp
