# mip-qtkj-layer

mip-qtkj-layer 点击显示隐藏

标题|内容
----|----
类型|通用
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|https://c.mipcdn.com/static/v1/mip-qtkj-layer/mip-qtkj-layer.js

## 示例

### 基本用法
```html
<mip-qtkj-layer sindex="0" cindex="0">
    自定义内容，可以嵌套其他组件
</mip-qtkj-layer>
```

## 属性

### sindex

说明：对应第几个显示按钮的角标
必选项：否

### cindex

说明：对应第几个显示内容的角标
必选项：否
