# mip-hs-rem

mip-hs-rem 组件说明
移动版rem相对屏幕适配
标题|内容
----|----
类型|通用
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|https://c.mipcdn.com/static/v1/mip-hs-rem/mip-hs-rem.js

## 示例

### 基本用法
```html
<mip-hs-rem base-width="750" scale-320="0.99" scale-362="1" scale-375="1" scale-414="0.97"></mip-hs-rem>
```
