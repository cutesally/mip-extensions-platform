# mip-jsh-count

mip-jsh-count 组件说明

标题|内容
----|----
类型|通用
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|https://c.mipcdn.com/static/v1/mip-jsh-count/mip-jsh-count.js

## 示例

### 基本用法
```html
<mip-jsh-count data-time='100'>
    自定义内容，可以嵌套其他组件
</mip-jsh-count>
```

## 属性

### data-time

说明：data-time 倒计时属性为剩余秒数
必选项：是
类型：int
取值范围：0-1000000
单位：无
默认值：0
