# mip-51la-lianyun

lianyun.51.la平台广告JS组件。

标题|内容
----|----
类型|通用
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|https://c.mipcdn.com/static/v1/mip-51la-lianyun/mip-51la-lianyun.js

## 示例

### 基本用法
```html
<mip-51la-lianyun id="APP_51LA_EDU" q="201">
    自定义内容，可以嵌套其他组件
</mip-51la-lianyun>
```

## 属性

### id

说明：广告类型ID
必选项：是
类型：字符串
默认值：无

### q

说明：广告位ID
必选项：是
类型：数字
默认值：无
